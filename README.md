# WhiteMAX Mod Android 7 Edition

### Предыстория

Мой старый телефон (honor 8) работает на android 7 без возможности OTA обновления, я не хочу перепрошивать его на android 8 (хотя рано или поздно придется) поскольку он уже тормозит на android 7. Поэтому я решил что будет отличной идеей бекпортить Максимку на android 7.

### Что не работает
- Не работает воспроизведение видео (звук есть)
- Не работает отправка видеособщений, голосовых и различных файлов (только текст и стикеры)
- Возможны прочие краши по приложению поскольку sdk 26 -> 24 слишком много изменений
